# UnityRestaurantCrush

## 유니티를 사용한 3-Match 퍼즐 게임 개발

### - FSM을 사용한 게임플레이 상태 관리
### - Simple Factory 패턴을 사용한 퍼즐 오브젝트 / 파티클 오브젝트 관리
### - Factory 패턴을 사용한 퍼즐 오브젝트 / 효과 오브젝트 관리
### - 콤보에 따른 폭탄, 라인제거 등 아이템 블록 생성 및 아이템 블록 사용 효과 구현
